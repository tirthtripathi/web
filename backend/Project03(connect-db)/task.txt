REST API 

GET /users - List all users in html document 

GET /api/users - List all users
GET /api/users/1 - Get data of user with id 1
GET /api/users/2 - Get data of user with id 2

POST /api/users/1 - Create user with id 1

PATCH /api/users/1 - Edit data of user with id 1

DELET /api/users/1 - Delet user with id 1
